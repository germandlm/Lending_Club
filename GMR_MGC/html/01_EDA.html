El objetivo de este trabajo es poder predecir si un cliente de Lending Club har un incumplimineto del pago de un crdito o no lo har. Para conseguir esto vamos a explorar el dataset, analizaremos los datos y probaremos varios modelos predictivos, considerando el mejor modelo el que mayor accuracy y menor overfitting tenga. El dataset con el que vamos a trabajar posee una serie de variables sobre individuos a los que se les ha concedido un crdito

  • Primero leemos los datos mostrando.
  • Eleccin de las variables ms importantes del conjunto de datos.
  • Filtramos los datos y realizamos el anlisis exploratorio.
  • Por ltimo, la realizacin de los modelos predictivos.
